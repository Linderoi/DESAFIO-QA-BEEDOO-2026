# Registro de Bugs

## Bug 1 – Cadastro permite campos vazios

**Passos para reproduzir**

1. Acessar tela de cadastro
2. Não preencher nenhum campo
3. Clicar em salvar

**Resultado atual**

O sistema permite salvar o curso.

**Resultado esperado**

O sistema deveria impedir o cadastro e exibir mensagem de erro.

**Severidade**

Alta

---

## Bug 2 – Campo descrição aceita texto ilimitado

**Passos**

1. Acessar cadastro
2. Inserir mais de 2000 caracteres
3. Salvar

**Resultado atual**

Campo aceita texto sem limite.

**Resultado esperado**

Deveria existir limite de caracteres.

**Severidade**

Média

---

## Bug 3 – Listagem não atualiza automaticamente

**Passos**

1. Cadastrar novo curso
2. Ir para listagem

**Resultado atual**

Curso não aparece imediatamente.

**Resultado esperado**

Listagem deveria atualizar automaticamente.

**Severidade**

Média
